# **App Name**: Navkar Living

## Core Features:

- Project Listings: Showcase featured real estate projects with images, location, and highlights.
- About Section: Display company information, mission, and history.
- Contact Form: Allow users to submit inquiries through a contact form.
- Brochure Access: Link to detailed brochures.
- AI-Powered Recommendations: Generative AI tool suggests properties matching user needs based on their message from the contact form.

## Style Guidelines:

- Primary color: Deep green (#184D2B), conveying trust and sophistication.
- Background color: Light green (#D6EAC3), providing a fresh, airy feel.
- Accent color: Seaweed green (#354A21), used sparingly to draw attention to key elements.
- Body and headline font: 'Inter', a sans-serif font known for its clean, readable design suitable for both headlines and body text.
- Use 'lucide-react' icons. Minimalist style, with consistent stroke weight and size.  Color should match primary color where appropriate.
- Grid-based layout with ample spacing to enhance readability and visual appeal.
- Subtle transitions and hover effects to add interactivity without being distracting.
